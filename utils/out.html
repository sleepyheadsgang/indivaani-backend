प्रयोग¶

In this section, demos on how to use all different integrated translators are provided.

टिप्पणी

You can always pass the languages by the name or by abbreviation.

उदाहरण: If you want to use english as a source or target language, you can pass अंग्रेज़ीयाएनएक तर्क के रूप में

आयात¶

से गहरा_अनुवादक आयात (गूगल अनुवादक,
                             पॉन्सट्रांसलेटर,
                             लिंगुईअनुवादक,
                             मायमेमोरीट्रांसलेटर,
                             यांडेक्सअनुवादक,
                             डीपएलट्रांसलेटर,
                             Qcriअनुवादक,
                             एकल_पहचान,
                             बैच_डिटेक्शन)

समर्थित भाषाओं की जाँच करें¶

टिप्पणी

You can check the supported languages of each translator by calling the get_supported_languages function as a static method.

# default return type is a list
langs_list = गूगल अनुवादक().समर्थित_भाषाएँ प्राप्त करें()  # output: [arabic, french, english etc...]

# alternatively, you can the dictionary containing languages mapped to their abbreviation
langs_dict = गूगल अनुवादक().समर्थित_भाषाएँ प्राप्त करें(as_dict=सत्य)  # output: {arabic: ar, french: fr, english:en etc...}

भाषा का पता लगाना¶

टिप्पणी

You can also detect language automatically. Notice that this package is free and my goal is to keep it free. Therefore, you will need to get your own api_key if you want to use the language detection function. I figured out you can get one for free here: https://detectlanguage.com/documentation

  • एकल पाठ का पता लगाना

लैंग = एकल_पहचान('bonjour la vie', एपीआई कुंजी='your_api_key')
छपाई(लैंग) # output: fr

  • बैच का पता लगाना

लैंग = बैच_डिटेक्शन(['bonjour la vie', 'hello world'], एपीआई कुंजी='your_api_key')
छपाई(लैंग) # output: [fr, en]

गूगल अनुवाद¶

मूलपाठ = 'happy coding'

  • You can use automatic language detection to detect the source language:

अनुवाद = गूगल अनुवादक(स्रोत='auto', लक्ष्य='de').अनुवाद(मूलपाठ=मूलपाठ)

  • You can pass languages by name or by abbreviation:

अनुवाद = गूगल अनुवादक(स्रोत='auto', लक्ष्य='german').अनुवाद(मूलपाठ=मूलपाठ)

# Alternatively, you can pass languages by their abbreviation:
अनुवाद = गूगल अनुवादक(स्रोत='en', लक्ष्य='de').अनुवाद(मूलपाठ=मूलपाठ)

  • पाठों के बैच का अनुवाद करें

ग्रंथों = ["hallo welt", "guten morgen"]

# the translate_sentences function is deprecated, use the translate_batch function instead
अनुवाद = गूगल अनुवादक('de', 'en').अनुवाद_बैच(ग्रंथों)

  • Translate from a file:

अनुवाद = गूगल अनुवादक(स्रोत='auto', लक्ष्य='german').Translate_फ़ाइल('path/to/file')

मायमेमोरी अनुवादक¶

टिप्पणी

As in google translate, you can use the automatic language detection with mymemory by using “auto” as an argument for the source language. However, this feature in the mymemory translator is not so powerful as in google translate.

  • सरल अनुवाद

मूलपाठ = 'Keep it up. You are awesome'

अनुवाद = मायमेमोरीट्रांसलेटर(स्रोत='auto', लक्ष्य='french').अनुवाद(मूलपाठ)

  • पाठों के बैच का अनुवाद करें

ग्रंथों = ["hallo welt", "guten morgen"]

# the translate_sentences function is deprecated, use the translate_batch function instead
अनुवाद = मायमेमोरीट्रांसलेटर('de', 'en').अनुवाद_बैच(ग्रंथों)

  • फ़ाइल से अनुवाद करें

पथ = "your_file.txt"

अनुवाद = मायमेमोरीट्रांसलेटर(स्रोत='en', लक्ष्य='fr').Translate_फ़ाइल(पथ)

डीपएलट्रांसलेटर¶

टिप्पणी

In order to use the DeeplTranslator translator, you need to generate an api key. Visit https://www.deepl.com/en/docs-api/ for more information

  • सरल अनुवाद

मूलपाठ = 'Keep it up. You are awesome'

अनुवाद = डीपएलट्रांसलेटर("your_api_key").अनुवाद(मूलपाठ)

  • पाठों के बैच का अनुवाद करें

ग्रंथों = ["hallo welt", "guten morgen"]

# the translate_sentences function is deprecated, use the translate_batch function instead
अनुवाद = डीपएलट्रांसलेटर("your_api_key").अनुवाद_बैच(ग्रंथों)

Qcriअनुवादक¶

टिप्पणी

In order to use the QcriTranslator translator, you need to generate a free api key. Visit https://mt.qcri.org/api/ for more information

  • भाषाओं की जाँच करें

# as a property
छपाई("language pairs: ", Qcriअनुवादक("your_api_key").बोली)

  • डोमेन जाँचें

# as a property
छपाई("domains: ", Qcriअनुवादक("your_api_key").डोमेन)

  • पाठ अनुवाद

मूलपाठ = 'Education is great'

अनुवाद = Qcriअनुवादक("your_api_key").अनुवाद(स्रोत='en', लक्ष्य='ar', कार्यक्षेत्र="news", मूलपाठ=मूलपाठ)
# output -> التعليم هو عظيم

# see docs for batch translation and more.

भाषाई अनुवादक¶

शब्द = 'good'

  • सरल अनुवाद

अनुवादित_शब्द = लिंगुईअनुवादक(स्रोत='english', लक्ष्य='french').अनुवाद(शब्द)

# pass language by their abbreviation
अनुवादित_शब्द = लिंगुईअनुवादक(स्रोत='en', लक्ष्य='fr').अनुवाद(शब्द)

  • मेल खाने वाले सभी पर्यायवाची या शब्द लौटाएँ

# set the argument return_all to True if you want to get all synonyms of the word to translate
अनुवादित_शब्द = लिंगुईअनुवादक(स्रोत='english', लक्ष्य='french').अनुवाद(शब्द, रिटर्न_ऑल=सत्य)

  • शब्दों के एक समूह का अनुवाद करें

अनुवादित_शब्द = लिंगुईअनुवादक(स्रोत='english', लक्ष्य='french').अनुवाद_शब्द(["good", "awesome"])

पोंस अनुवादक¶

टिप्पणी

You can pass the languages by the name or by abbreviation just like previous examples using GoogleTranslate

शब्द = 'awesome'

  • सरल अनुवाद

अनुवादित_शब्द = पॉन्सट्रांसलेटर(स्रोत='english', लक्ष्य='french').अनुवाद(शब्द)

# pass language by their abbreviation
अनुवादित_शब्द = पॉन्सट्रांसलेटर(स्रोत='en', लक्ष्य='fr').अनुवाद(शब्द)

  • मेल खाने वाले सभी पर्यायवाची या शब्द लौटाएँ

# set the argument return_all to True if you want to get all synonyms of the word to translate
अनुवादित_शब्द = पॉन्सट्रांसलेटर(स्रोत='english', लक्ष्य='french').अनुवाद(शब्द, रिटर्न_ऑल=सत्य)

  • शब्दों के एक समूह का अनुवाद करें

अनुवादित_शब्द = पॉन्सट्रांसलेटर(स्रोत='english', लक्ष्य='french').अनुवाद_शब्द(["good", "awesome"])

यांडेक्स अनुवादक¶

टिप्पणी

आपको n की आवश्यकता हैनिजी एपीआई कुंजी if you want to use the yandex translator. Visit the official website for more information about how to get one

  • भाषा का पता लगाना

लैंग = यांडेक्सअनुवादक('your_api_key').पता लगाना('Hallo, Welt')
छपाई(f"language detected: {लैंग}")  # output -> language detected: 'de'

  • पाठ अनुवाद

# with auto detection | meaning provide only the target language and let yandex detect the source
अनुवाद = यांडेक्सअनुवादक('your_api_key').अनुवाद(स्रोत="auto", लक्ष्य="en", मूलपाठ='Hallo, Welt')
छपाई(f"translated text: {अनुवाद}")  # output -> translated text: Hello world

# provide source and target language explicitly
अनुवाद = यांडेक्सअनुवादक('your_api_key').अनुवाद(स्रोत="de", लक्ष्य="en", मूलपाठ='Hallo, Welt')
छपाई(f"translated text: {अनुवाद}")  # output -> translated text: Hello world

  • फ़ाइल अनुवाद

अनुवाद = यांडेक्सअनुवादक('your_api_key').Translate_फ़ाइल(स्रोत="auto", लक्ष्य="en", पथ="path_to_your_file")

  • बैच अनुवाद

अनुवाद = यांडेक्सअनुवादक('your_api_key').अनुवाद_बैच(स्रोत="auto", लक्ष्य="de", बैच=["hello world", "happy coding"])

लिब्रे अनुवादक¶

टिप्पणी

लिब्रे अनुवाद में एकाधिक हैंदर्पण which can be used for the API endpoint. Some require an API key to be used. By default the base url is set to libretranslate.de . This can be set using the “base_url” input parameter.

मूलपाठ = 'laufen'
अनुवाद = लिबरट्रांसलेटर(स्रोत='auto', लक्ष्य='en', बेस_यूआरएल = 'https://libretranslate.com/', एपीआई कुंजी = 'your_api_key').अनुवाद(मूलपाठ=मूलपाठ)  # output: run

  • You can pass languages by name or by abbreviation:

अनुवाद = लिबरट्रांसलेटर(स्रोत='german', लक्ष्य='english').अनुवाद(मूलपाठ=मूलपाठ)

# Alternatively, you can pass languages by their abbreviation:
अनुवाद = लिबरट्रांसलेटर(स्रोत='de', लक्ष्य='en').अनुवाद(मूलपाठ=मूलपाठ)

  • पाठों के बैच का अनुवाद करें

ग्रंथों = ["hallo welt", "guten morgen"]
अनुवाद = लिबरट्रांसलेटर(स्रोत='auto', लक्ष्य='en').अनुवाद_बैच(ग्रंथों)

  • Translate from a file:

अनुवाद = लिबरट्रांसलेटर(स्रोत='auto', लक्ष्य='en').Translate_फ़ाइल('path/to/file')

टर्मिनल से उपयोग¶

For a quick access, you can use the deep_translator from terminal. For this to work, you need to provide the right arguments, which are the translator you want to use, source language, target language and the text you want to translate.

For example, provide “google” as an argument to use the google translator. Alternatively, you can use the other supported translators. Just read the documentation to have an overview about the supported translators in this library.

$ गहरा_अनुवादक --अनुवादक "google" --स्रोत "english" --लक्ष्य "german" --मूलपाठ "happy coding"

Or you can go for the short version:

$ गहरा_अनुवादक -ट्रांस "google" -src "english" -टीजी "german" -TXT "happy coding"

If you want, you can also pass the source and target language by their abbreviation

$ गहरा_अनुवादक -ट्रांस "google" -src "en" -टीजी "de" -TXT "happy coding"
